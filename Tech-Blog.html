The DOM is an API (Application Programming Interface). Its is the skeleton or blueprint of the webpage. The DOM Tree on the left shows a very basic webpage DOM. The terms 'Parent', 'Child' and 'Sibling' are used to recognise the elements.

The oldest Parent is the document here, with the Root Element the child This would make the Head and Body Elements the 'Grandchildren' of the Doument. These again have children for the head the title and then the 'div', 'p', 'a' and 'span' are all child elements of the body.

elements that share a parent can be refeered to as siblings, the Head and Body would be siblingsas would the 'div', 'h1', 'p', 'a', 'span'.
